# Historical Badminton Performance Analytics (2023–2026)

## Project Overview

This project analyzes badminton match performance data collected between 2023 and 2026. The objective was to identify trends in participation, performance, tournament progression, event-wise results, and category-wise achievements through data analysis and visualization.

## Dataset Information

* Total Matches Analyzed: 40
* Time Period: 2023–2026
* Event Types: Singles, Doubles, Mixed Doubles
* Categories: U-15, U-17, U-19

## Tools Used

* Google Sheets
* Spreadsheet Formulas
* Data Visualization
* Dashboard Design

## Skills Demonstrated

* Data Cleaning
* Data Analysis
* KPI Analysis
* Trend Analysis
* Sports Analytics
* Dashboard Development
* Report Writing

## Key Findings

* Overall Win Rate: 55%
* Most Played Event: Singles
* Strongest Category: GS U-17
* Highest Activity Year: 2024
* Reached Final Stage: 2 Times

## Project Structure

1. Project Overview
2. Summary Dashboard
3. Match Data
4. Event Analysis
5. Category Analysis
6. Tournament Progression Analysis
7. Performance Growth Analysis

## Conclusion

This project demonstrates how spreadsheet-based analytics can be used to evaluate badminton performance across multiple dimensions and generate meaningful insights from real competitive data.
